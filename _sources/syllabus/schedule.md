A list of topics we will cover can be found below. Note that topics may be subject to change.

- Regression
- Gradient descent
- Classification
- Model selection and evaluation
- Regularization
- Feature engineering
- Fairness and interpretability
- Ensembling
- Neural networks
- Unsupervised learning
    - K-means clustering
    - Autoencoders
