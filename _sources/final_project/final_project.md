# Final Project

# Learning objectives

- Apply the machine learning principles you have learned in an area of your choice
- Follow the machine learning process from start to finish independently
- Gain experience communicating scientific results:
    - Visually and in writing through an Jupyter Book article
    - Verbally through a brief teaching presentation to the class

# Logistics and assessment breakdown

- Final project groups can be one to three students.
- Students are expected to contribute equally to the project, and the amount of work should scale with the size of the groups.

# Components of the report

- Datasheet: a description of the dataset you used, including the source, the variables, and the units of analysis.
- ML task formulation: a description of the machine learning task you are trying to solve, including the target variable, the features, and the type of machine learning model you are using.
- ML development process narrative
- Model card for model reporting on your best performing model
